# AI-Assisted Development

This project was structured to make AI assistance reviewable:

- Requirements were converted into explicit schemas, state transitions, tests, and docs.
- Deterministic local providers make behavior repeatable for local verification.
- Generated report text is constrained to confirmed facts.
- Tests cover validation, suggestion review, grading flow, report generation, finalization, and audit events.

Responsible use means treating AI output as draft material, then validating it with types, tests, runtime schemas, and human review.
