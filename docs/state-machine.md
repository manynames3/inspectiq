# State Machine

Statuses:

`DRAFT`, `NEEDS_PHOTOS`, `READY_FOR_GRADING`, `GRADED`, `AI_DRAFT_PENDING`, `AI_DRAFTED`, `HUMAN_REVIEW_REQUIRED`, `FINALIZED`, `REPORT_FAILED`.

Allowed transitions:

- `DRAFT -> NEEDS_PHOTOS`
- `DRAFT -> READY_FOR_GRADING`
- `NEEDS_PHOTOS -> READY_FOR_GRADING`
- `READY_FOR_GRADING -> GRADED`
- `GRADED -> AI_DRAFT_PENDING`
- `AI_DRAFT_PENDING -> AI_DRAFTED`
- `AI_DRAFT_PENDING -> HUMAN_REVIEW_REQUIRED`
- `AI_DRAFT_PENDING -> REPORT_FAILED`
- `REPORT_FAILED -> AI_DRAFT_PENDING`
- `AI_DRAFTED -> HUMAN_REVIEW_REQUIRED`
- `AI_DRAFTED -> FINALIZED`
- `HUMAN_REVIEW_REQUIRED -> FINALIZED`
- `HUMAN_REVIEW_REQUIRED -> AI_DRAFT_PENDING`

`FINALIZED` is terminal for normal workflow. The API enforces these transitions in `apps/api/src/stateMachine.ts`.
