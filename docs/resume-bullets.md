# Resume Bullets

## Short

- Built InspectIQ, a React/TypeScript, Node.js, Java, and Postgres-modeled portfolio app for AI-assisted vehicle inspection workflows with human review, deterministic grading, audit trails, tests, and AWS-ready architecture.

## Technical

- Developed a full-stack inspection platform using React, TypeScript, Express, Zod, Java Spring Boot grading rules, Postgres schema/Drizzle definitions, S3-style image metadata, Step Functions-style async report jobs, Bedrock/Claude provider abstractions, CI, and integration tests.

## Business-impact

- Modeled an enterprise inspection and imaging workflow that turns vehicle photos into reviewable AI suggestions, requires human confirmation before facts affect condition grades, generates schema-validated report drafts, and preserves an audit trail for operational accountability.
